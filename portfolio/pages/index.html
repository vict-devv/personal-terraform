Linkr — URL Shortener Platform WIP

linkr link-tf

Self-hosted URL shortening platform built in Go, composed of three independent microservices communicating via RabbitMQ. The shortener-api handles URL creation and resolution with a Redis read-through cache backed by Postgres. On every redirect, it emits a redirect.clicked event consumed by the analytics-worker, which persists click data to MongoDB. A separate read-only stats-api exposes aggregated analytics (total clicks, daily trends, top referrers) for dashboards. Infrastructure is fully provisioned with Terraform on AWS EKS, with per-environment state (dev/prod) stored in S3.

Go • RabbitMQ • Redis • Postgres • MongoDB • Docker • AWS EKS • Terraform • VPC
